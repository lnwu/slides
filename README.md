# MCP & Custom Agent Instructions Slides

A presentation about Model Context Protocol (MCP) and how to customize agent/plan mode instructions.

Built with [Slidev](https://sli.dev/).

## Getting Started

```bash
# Install dependencies
npm install

# Start the slide show
npm run dev

# Build for production
npm run build

# Export to PDF
npm run export
```

## Content

1. **What is MCP?** - Introduction to Model Context Protocol
2. **MCP Architecture** - How MCP connects AI to data sources
3. **Customizing Agent Instructions** - Tailoring AI behavior
4. **Plan Mode vs Agent Mode** - Understanding different modes
5. **Best Practices** - Tips for effective customization
